# A Simple Client Rolodex

  created in with React and Javascript, 

## How to install

  open terminal / console:

### `npm install`
  installs all packages and dependencies needed to run
  
### `npm start`
  npm start in terminal to run Client Rolodex app in the development mode.\
  Open [http://localhost:3000] in your browser to view in a local environment

## Usage

### type client name into search bar and watch your results filter in real time
